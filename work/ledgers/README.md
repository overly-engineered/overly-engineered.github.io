Ionic Ledgers
=====================

## Using this project

Install Ionic

```bash
$ npm install -g ionic
```

Then run:

```bash
$ cd ledgers
$ ionic serve
```

More info on this can be found on the Ionic [Getting Started](http://ionicframework.com/getting-started) page and the [Ionic CLI](https://github.com/driftyco/ionic-cli) repo.
