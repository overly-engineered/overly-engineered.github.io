isNan

There are two different versions of isNan that produce different results

isNan()

The global isNan function produces a lot of false positives

isNaN(false); // false
isNaN(true); // false
isNaN(0); // false
isNaN("string"); // true
isNaN({}); // true
isNaN(NaN); // true

Number.isNaN()

Number.isNaN is more reliable at detecting what is NaN and what isnt.

isNaN(false); // false
isNaN(true); // false
isNaN(0); // false
isNaN("string"); // false
isNaN({}); // false
isNaN(NaN); // true
